## What I Built

**Code-Collab** is a Collaborative IDE that enables real-time code collaboration and development. This project showcases expertise in:

- **Frontend:** React, TypeScript, Figma design integration
- **Backend:** Next.js, Vercel deployment
- **DevOps & Tools:** Docker containerization, GitHub Actions CI/CD
- **Programming Languages:** TypeScript, Python, C#, and more

The application demonstrates full-stack development capabilities with a focus on creating seamless collaborative experiences for developers.

---

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
